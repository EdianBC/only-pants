# only-pants

![only-pants icon](icon300.png)

Do you hate that short video format like TikToks, Reels and Shorts? Well, I don't care, I do. So I got rid of them and their tempting, clickbait miniatures designed to waste my precious time (which I would rather waste another way).

To use this extension locally on Microsft Edge just type `edge://extensions/` in the address bar, activate Developer Mode and click on the "Load Unpacked" button that will appear.
